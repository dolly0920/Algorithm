2×n 크기의 직사각형을 1×2, 2×1 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오.

아래 그림은 2×5 크기의 직사각형을 채운 한 가지 방법의 예이다.
----------------------------------------------------------------------------------------------------------------
//My sol
#include <stdio.h>

int N;
int d[1000 + 10];

int dp(int n)
{
	if (n == 1)	return 1;
	if (n == 2)	return 2;
	if (d[n] != 0)	 return d[n];
	return d[n] = (dp(n - 1) + dp(n - 2)) % 10007;
}

int main(void)
{
	scanf("%d", &N);

	printf("%d", dp(N));


	return 0;
}
